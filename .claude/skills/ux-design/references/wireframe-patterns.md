# ASCII 와이어프레임 패턴

OnVoy 화면을 표현할 때 자주 쓰는 패턴 모음. 구조와 우선순위를 명확히 보여주는 것이 목적이다.

## 모바일 페이지 표준 골격

```
┌─────────────────────────────┐
│ ⬆ Safe Area Top             │
├─────────────────────────────┤
│ ← [헤더 타이틀]      [액션] │  헤더: 15px bold, brand.ink
├─────────────────────────────┤
│                             │
│ [본문 영역]                  │  여백 1.5배 (Whitespace)
│                             │
├─────────────────────────────┤
│ [Bottom Sheet / Tab Bar]    │
├─────────────────────────────┤
│ ⬇ Safe Area Bottom          │
└─────────────────────────────┘
```

## 카드 컴포넌트

```
┌─────────────────────────────┐  ← 12px radius, shadow.sm
│ 제목 (15px bold, ink)        │
│ 부가정보 (13px, muted)       │
│                             │
│ [태그] [태그]   [Primary →] │  ← Cobalt Blue는 우측 액션만
└─────────────────────────────┘
```

## 모달 / Bottom Sheet

```
        ┌─────────────────────┐
        │ ━━━ (drag handle)  │  ← 모달 16px radius
        ├─────────────────────┤
        │ 제목                │
        │                    │
        │ [본문]             │
        │                    │
        │ ┌─────────────────┐│
        │ │  Primary CTA    ││  ← Cobalt Blue 채움
        │ └─────────────────┘│
        └─────────────────────┘
        scrim (bg.scrim @0.5)
```

## 체크리스트 (아코디언)

```
┌─────────────────────────────┐  ← 16px radius (메인 컨테이너급)
│ ▼ 그룹명 (15px bold)    3/5 │  ← 진행률은 텍스트로만
├─────────────────────────────┤
│ ☐ 항목 1 (15px)             │  ← 평상시 액션 없음 (노이즈 0)
│ ☑ 항목 2 (15px, line-thru)  │
│ ☐ 항목 3 (15px)             │
└─────────────────────────────┘
       ← 스와이프 시 [수정] [삭제] 노출
```

## 타임라인 (일정표)

```
│●━━━━ 09:00 도쿄 도착          │  ← ● Cobalt Blue (선택/현재)
│ │
│○──── 11:00 호텔 체크인        │  ← ○ brand.hairline
│ │
│○──── 14:00 신주쿠 이동        │
```

## 폼 입력

```
이메일
┌─────────────────────────────┐  ← 8px radius
│ you@example.com             │  ← brand.hairlineSoft border
└─────────────────────────────┘

비밀번호                    [표시]
┌─────────────────────────────┐
│ ••••••••                    │
└─────────────────────────────┘

⚠ 비밀번호는 8자 이상이어야 합니다  ← 에러는 ⚠ + 텍스트 (색만으로 X)
```

## 빈 상태 (Empty State)

```
        ┌─────┐
        │ 🧳  │   ← 일러스트/아이콘
        └─────┘
       
   아직 여행이 없어요
   첫 여행을 만들어보세요
   
   ┌──────────────────┐
   │  새 여행 만들기  │   ← Primary CTA 만 색
   └──────────────────┘
```

## 작성 팁

- 시각 정밀도가 아니라 **구조와 우선순위**를 보여준다
- 각 영역마다 **사용 토큰**을 주석으로 표시 (`brand.ink`, `12px radius` 등)
- **인터랙션**은 화살표/주석으로 (`→ 스와이프 시...`)
- **데이터 출처**도 표시 가능 (`← Trip.startDate 포맷`)
